Getting Started

Sign Up

As a visitor u don't need to sign up. But to post auction as an auctioneer or to participate in bidding as a bidder creating account is the first step. Auctioneer account can be either private or government. For the latter an admin will approve, if the admin found the account to be authentic or truly represents government. In the case of private auctioneer the admin will approve if its very active auctioneer.

Logging In

After creating account a user can login as a bidder or auctioneer. If the user forgets password he or she must enter the phone they submitted while creating account. Then they will receive a code to their phone. They have to fill that code to create new password. Submitting auction

Posting An Auction

A user must create an auctioneer account. And then choose profile auctioneer. When the user fills auction form he or she should bear in mind that after submitting the auction, payment for the auction is a necessity to make the auction available for bidding. After posting auction auctioneer can remove the auction.

Participating In Auction

The bidding process concern solely Bidders. But inorder to bid, a bidder must log in. Then a bidder must subscribe to submit bid. If the auction is normal there is a minimum starting bid. So bidder must bid more than that. If the auction is reverse there is a maximum bid of which bidder can not submit more than that. After auction ends subscribed bidders can the expired auction.
